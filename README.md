## 🧠 Features

✨ **Auto-analysis at your fingertips:**

- 📁 Upload `.csv` or `.xlsx` files  
- 🧮 Auto-detect data types, nulls, and basic structure  
- 📋 Descriptive statistics summary  
- 🔥 Correlation matrix and heatmap  
- 📊 Interactive histograms using Plotly  
- 🖼 Built with **Streamlit** for a sleek GUI  
- ☁️ One-click deploy on **Streamlit Cloud**
